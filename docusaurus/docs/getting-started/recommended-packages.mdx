title: Recommended Packages
---

Some packages are so common they'll be included by default, while others are more project specific, but still highly recommended for their use cases.

## react-table

- [Docs](https://swr.vercel.app/)

Data modification for tables. Bifrost's Table component is only responsible for the view, and will not do any data modification for you.

## swr

- [Docs](https://react-table.tanstack.com/)

Data Fetching library using React hooks. Makes data fetching easy, comes with a cache, Suspense support and more.
