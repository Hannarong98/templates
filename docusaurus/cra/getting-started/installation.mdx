## Prerequisites

- Some experience with React
- [node.js](https://nodejs.org/)
- [Visual Studio Code](https://code.visualstudio.com/) with the following plugins
  - [Node Extension Pack](https://marketplace.visualstudio.com/items?itemName=swellaby.node-pack)
  - [Prettier](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode)
- Set up the Intility NPM registry (optional)

### Intility NPM Registry (optional)

The Intility NPM registry (includes access to `@intility/*` and `@fortawesome/*` packages) can be configured by running:

```bash
npm config set registry https://artifactory.int.intility.no/artifactory/api/npm/npm/
```

For other alternatives, see the [guide on developers.intility.no](https://developers.intility.no/guides/tutorials/npm) (login required).

## Create App

Select the version you want. By default, TypeScript and an authentication sample is included.

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs
  groupId="typed"
  defaultValue="typed"
  values={[
    { label: 'TypeScript', value: 'typed' },
    { label: 'JavaScript', value: 'plain' },
  ]}
>
<TabItem value="typed">

```bash
npx create-react-app my-app --template "@intility"
```

</TabItem>
<TabItem value="plain">

```bash
npx create-react-app my-app --template "@intility/js"
```

</TabItem>
</Tabs>

Run it in your terminal, and then open the Visual Studio Code Workspace with the following command.

```bash
code my-app/app.code-workspace
```

You can now start coding by running the start script.

```bash
npm start
```
